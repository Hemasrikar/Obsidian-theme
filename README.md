# Obsidian-theme

This is Repository contains custom theme made to my liking based on [Neutral Academia Theme](https://github.com/incantatem2/Obsidian-neutral-academia), CSS scripts and plugins for my personal vault.

The [Obsidian-Legacy-Vault-Switcher](https://github.com/Hemasrikar/Obsidian-theme/tree/main/Obsidian-Legacy-Vault-Switcher) is a plugin to move the valut, settings bar on to the left sidebar.

This is my primary theme for my academic valut. I will add features or adjust the theme as I go. If you want to use this theme, just download theme.css file and mainfest.json file, place them in a folder and place that folder in the obsidian theme folder.
